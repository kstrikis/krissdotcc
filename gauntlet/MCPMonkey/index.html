MCPMonkey

MCPMonkey is a fork of Violentmonkey that extends userscript capabilities to support Model Context Protocol (MCP) servers. It effectively bridges the gap between AI language models and browser interactions, enabling advanced AI tools to have deeper integration with web browsing.

The project features enhanced browser connectivity for AI tools, providing capabilities like tab management, page metadata extraction, and page style information retrieval for more contextually aware AI assistance.

MCPMonkey is designed to work with Cursor and other AI-driven development environments, improving the way AI tools can interact with and understand web content for more effective assistance and automation.
