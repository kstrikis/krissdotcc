ReelAI

ReelAI is a native iOS application built with Swift that transforms raw ideas into engaging, AI-generated videos designed to captivate attention and spark viral meme culture. The platform delivers a robust pipeline for creating, processing, and sharing video content that ranges from hilariously absurd to artlessly baffling—ensuring every piece of content is engineered to provoke a reaction.

The application is built on a Firebase Ecosystem integrated to manage user authentication, real-time data syncing, media asset storage, and performance monitoring. Key features include basic in-app video recording and upload, real-time playback feed, and storymaker. Storymaker allows users to create full multi-scene, narrated, AI generated videos with sound effects by providing a short text prompt.

Implemented with Swift and Apple's Combine framework for clean, asynchronous code, with compute-intensive tasks (video rendering, AI processing) offloaded to Firebase Cloud Functions and external APIs, ensuring scalability and high uptime.
