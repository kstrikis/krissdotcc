sana-mobile

Sana Mobile is a React Native mobile application for both Android and iOS platforms, providing Sana members with access to their healthcare providers and insurance information.

Key features include secure authentication with PIN and biometric options, chat with healthcare providers, completing questionnaires, and viewing insurance information and benefits.

The tech stack includes React Native with Expo, TypeScript, React Navigation, Expo Secure Store for secure data storage, Expo Local Authentication for biometric authentication, and Axios for API requests, all designed with HIPAA-compliant security measures in place.
