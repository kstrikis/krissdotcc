AutoCRM

AutoCRM is a modern, AI-powered Customer Relationship Management platform that minimizes human workload in customer support while enhancing the customer experience. By leveraging generative AI and existing help resources, AutoCRM delivers an interactive support experience with minimal human involvement.

Key features include AI-powered ticket resolution and management with Supabase and enterprise-grade authentication with Supabase Auth.

The tech stack includes React with TypeScript, Tailwind CSS for styling, Supabase for database, auth, and real-time features, Supabase Edge Functions for backend logic, and structured output from OpenAI via LangChain including tracability in LangSmith.
